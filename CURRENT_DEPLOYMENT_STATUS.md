## デプロイを維持する方法

### 自動デプロイ（推奨）

GitHubリポジトリがVercelに接続されている場合、以下の操作で自動的にデプロイが維持されます：

1. **GitHubにプッシュするだけで自動デプロイ**
   ```bash
   git push origin main
   ```
   - これで自動的にVercelにデプロイされます

2. **Vercelダッシュボードで確認**
   - https://vercel.com にログイン
   - プロジェクト（VMEDA）を開く
   - 「Deployments」タブでデプロイ状態を確認

### 手動デプロイ

必要に応じて、Vercel CLIで手動デプロイも可能：

```bash
vercel --prod
```
