# プライバシーポリシー

## 個人情報の収集について

このサイト（VMEDA）は、**個人を特定できる情報を収集しません**。

### 収集していない情報

- **IPアドレス**: 収集・保存していません
- **クッキー**: 使用していません
- **トラッキングスクリプト**: Google Analyticsなどのトラッキングツールは使用していません
- **ブラウザフィンガープリンティング**: 使用していません
- **位置情報**: 収集していません
- **個人を特定できるその他の情報**: 収集していません

### 収集している情報

- **検索ワードのみ**: 検索履歴として保存されますが、検索ワードのみです（個人を特定できません）

### サーバーログについて

サーバー側のログには以下の情報が記録される場合がありますが、これらは**デバッグ目的のみ**で使用され、個人を特定するために使用されることはありません：

- **User-Agent**: ブラウザの種類を識別するため（例: "iPhone", "Chrome"）
- **リクエストパス**: どのページにアクセスしたか（例: "/api/search"）

これらの情報は：
- 個人を特定するために使用されません
- 第三者に共有されません
- 一定期間後に自動的に削除されます（Vercelのログ保持ポリシーに従います）

### 検索履歴について

- 検索履歴には**検索ワードのみ**が保存されます
- IPアドレスやその他の個人情報は含まれません
- 検索履歴は最新30件のみ保持され、古いものは自動的に削除されます
- 検索履歴はすべてのユーザーに公開されます（検索ワードのみ）

### データの保存場所

- 検索履歴はMongoDB Atlasに保存されます
- MongoDB Atlasは暗号化された接続を使用します
- データはMongoDB Atlasのセキュリティポリシーに従って保護されます

### 第三者への情報提供

このサイトは、以下の場合を除き、個人情報を第三者に提供することはありません：

- 法的な要求がある場合（裁判所の命令など）
- サイトの運営に必要な場合（ホスティングサービスなど）

### プライバシーの保護

このサイトは、ユーザーのプライバシーを尊重し、個人を特定できる情報を収集しないように設計されています。

## 技術的な詳細

### 実装の確認

現在の実装では：

1. **IPアドレスの収集を削除済み**
   ```javascript
   // 以前: const searchEntry = { query, timestamp, ip: req.ip };
   // 現在: const searchEntry = { query }; // IPアドレスは含まれません
   ```

2. **クッキーを使用していない**
   - `cookie-parser`などのミドルウェアは使用していません

3. **トラッキングスクリプトを使用していない**
   - Google Analytics、Facebook Pixelなどのトラッキングツールは使用していません

4. **ブラウザストレージを使用していない**
   - `localStorage`、`sessionStorage`は使用していません

5. **フィンガープリンティングを使用していない**
   - Canvas、WebGL、AudioContextなどのフィンガープリンティング技術は使用していません

### Vercelのログについて

Vercelのインフラレベルでは、以下の情報がログに記録される場合があります：

- **IPアドレス**: Vercelのインフラが自動的に記録（これはVercelの機能であり、アプリケーションコードでは制御できません）
- **リクエストログ**: アクセスログ

これらのログは：
- Vercelのダッシュボードでのみ閲覧可能です
- アプリケーションコードからはアクセスできません
- Vercelのプライバシーポリシーに従って管理されます

### 結論

このサイトは、**個人を特定できる情報を収集・保存していません**。検索ワードのみが保存され、すべてのユーザーに公開されます。個人のプライバシーを保護する設計になっています。
